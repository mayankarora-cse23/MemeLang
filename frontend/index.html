// 02 SYNTAX
The Sacred Keywords

All programs must start with padh le bhai and end with main chala. Anything outside is ignored (like your attendance).

padh le bhai
Program Start
like int main() but with more regret
main chala
Program End
you survived. barely.
yaar ye hai
Variable Declaration
equivalent to var/let
bol yaar
Print / Output
printf but with feelings
sun yaar
Input (stdin)
listening (unlike your prof)
scene hai
true (boolean)
it's valid, it's happening
scene nahi
false (boolean)
nope. rejected. denied.
null pointer tha
null / nil
the classic BTech villain
agar scene ho
if condition
if things are looking good
warna yaar
else
plan B (also didn't work)
nahi to scene
else if
middle child of conditionals
jab tak grind
while loop
grind until it works
bas kar yaar
break
I quit. I'm done. goodbye.
skip kar
continue
like skipping 8am lecture
kaam karo
function declaration
do your job, please
lele result
return value
here's your output, go away
// bhai sun
Single-line comment
inline annotation, ignored by compiler
/* ruk yaar */
Multi-line comment
long explanation nobody reads
